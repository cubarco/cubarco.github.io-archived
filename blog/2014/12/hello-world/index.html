    By Cubarco December 16, 2014 January 20, 2016 Comment Tweet Like

    之前挂 wordpress 的 vps 快到期了，有 do 免费用都不想续费了。一直觉得 wordpress 很臃肿，刚好趁这次机会把 blog 换成 jekyll。

    今天把 / 和 /blog/ 都放在 github 上，反正都是静态页面，没必要 vps 处理。然后用 cloudfare 做了 cdn, 这个站已经和 vps 没关系了。
